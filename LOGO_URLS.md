# Gruby Logo URLs

All logo assets are hosted on Vercel and publicly accessible. Use these URLs for API integrations, press kits, and external services.

## Base URL
```
https://gruby.app
```

## Logo Assets

### Icon PNG (Recommended for APIs)
**Best for:** Kroger API, app stores, social media, favicons
```
https://gruby.app/logos/gruby-icon.png
```
- Format: PNG
- Size: 1024x1024
- File size: ~22KB

### Icon SVG
**Best for:** Web, scalable displays, print
```
https://gruby.app/logos/gruby-icon.svg
```
- Format: SVG
- Size: Scalable
- File size: ~2.4KB

### Full Logo SVG

## Usage Guidelines

✅ **Do:**
- Use on white or light backgrounds
- Maintain minimum clear space
- Keep proportional when resizing
- Use icon for small spaces
- Use full logo for headers

❌ **Don't:**
- Distort or stretch the logo
- Change the colors
- Place on busy backgrounds
- Rotate or tilt the logo
- Add effects or filters
